# Banya Gallery Images

Please add the following 6 WebP images to this directory:

1. **banya-steam-room.webp** - Photo of the steam room/парная
2. **banya-rest-area.webp** - Photo of the rest area/зона отдыха
3. **banya-cold-pool.webp** - Photo of the cold pool/холодный бассейн
4. **banya-hot-stones.webp** - Photo of hot stones/горячие камни
5. **banya-tea-ceremony.webp** - Photo of tea ceremony area/чайная церемония
6. **banya-lounge.webp** - Photo of the lounge area/лаунж зона

## Optimization recommendations:
- Format: WebP
- Resolution: 1200x900px or similar aspect ratio (4:3)
- Quality: 80-85% for optimal file size
- File size: Try to keep under 200KB per image

The gallery component will automatically load these images from:
`/images/banya/gallery/[filename].webp`
