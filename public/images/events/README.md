# Event Images Directory

Place your event poster images here:

## Required Image:
- `sauna-rave-poster.jpg` - The SAUNA RAVE event poster

## Image Guidelines:
- Recommended format: JPG or WebP
- Optimal dimensions: 800x1200px (portrait) or 1200x800px (landscape)
- File size: Keep under 500KB for optimal loading
- Quality: 80-85% JPEG quality is recommended

## Naming Convention:
- Use lowercase with hyphens
- Include event name and date if relevant
- Example: `sauna-rave-poster.jpg`, `event-name-2025-01.jpg`
