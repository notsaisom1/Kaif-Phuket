# คู่มือพนักงาน — แก้เนื้อหาเว็บ KAIF

ใช้หน้าแอดมิน (Sanity Studio) เพื่อเปลี่ยนข้อมูลบนเว็บโดยไม่ต้องเขียนโค้ด

## เข้าใช้งาน

1. เปิดลิงก์ Studio ที่ทีมส่งให้ (เช่น `https://kaif-phuket.sanity.studio`)
2. Login ด้วยอีเมลที่ได้รับเชิญ
3. แก้ข้อมูลแล้วกด **Publish**
4. เปิดเว็บ [kaif-phuket.com](https://kaif-phuket.com) รีเฟรชหน้า — ข้อมูลจะอัปเดต (อาจใช้เวลาสักครู่)

---

## 1) เบอร์โทร / WhatsApp / ที่อยู่ / เวลา

เมนูซ้าย → **Site Settings**

| ช่อง | ใส่อะไร |
|------|---------|
| Phone (display) | เบอร์ที่แสดง เช่น `+66 62 480 5877` |
| Phone (tel: link) | เบอร์สำหรับกดโทร เช่น `+66624805877` |
| WhatsApp number | ตัวเลขล้วน ไม่มี `+` เช่น `66624805877` |
| Address | ที่อยู่ (อังกฤษ / รัสเซีย / ไทย) |
| Opens at / Closes at | เช่น `07:00` และ `21:00` |
| Hours label | (ถ้าต้องการ) ข้อความเต็ม เช่น `ทุกวัน: 07:00 – 21:00` |

กด **Publish** เมื่อแก้เสร็จ

---

## 2) อีเวนต์

เมนูซ้าย → **Events** → **Create**

- ใส่ชื่อ / คำอธิบาย ทั้ง EN / RU / TH
- อัปโหลดรูปโปสเตอร์
- ใส่วันที่ (Day, Month เช่น `MAR`, Full date)
- เปิด **Published** เพื่อให้โชว์บนเว็บ
- ปิด Published = ซ่อนจากเว็บโดยไม่ลบ

---

## 3) โปรโมชันรายวัน

เมนูซ้าย → **Promotions**

- แก้ชื่อ คำอธิบาย สี
- อัปโหลดรูปใหม่ (หรือใช้ path เดิมใน Fallback image path)
- ปิด **Active** ถ้าอยากซ่อนโปรโมนั้น

---

## 4) รูปสำคัญ (Hero / แบนเนอร์)

เมนูซ้าย → **Key Images**

- เลือก key เช่น `homeHero` หรือ `promoBanner`
- อัปโหลดรูปใหม่ → Publish

> หมายเหตุ: รูปเหล่านี้จะโชว์บนเว็บเมื่อหน้าเว็บเชื่อม key นั้นแล้ว หากยังไม่เห็น แจ้งทีมพัฒนา

---

## ข้อควรระวัง

- ต้องกด **Publish** ทุกครั้ง ไม่งั้นเว็บไม่เปลี่ยน
- อย่าลบเอกสาร **Site Settings**
- รูปควรไม่ใหญ่เกิน (~2–5 MB) และชัดบนมือถือ
- ถ้าไม่แน่ใจ ให้แก้แล้วเช็คเว็บจริงก่อนส่งต่อลูกค้า
